## How to run
1. Clone the repo.
2. Ensure Python 3.x and required packages are installed:
   ```bash
   pip install pandas numpy scipy scikit-learn matplotlib seaborn
   ```
3. Run scripts; example:
   ```bash
   python Week-5-EDA/eda_script.py
   python Week-6-Statistical_Testing/statistical_test.py
   python Week-7-ML_Introduction/linear_regression_model.py
   python Week-8-Model_Evaluation/model_tuning.py
   ```
